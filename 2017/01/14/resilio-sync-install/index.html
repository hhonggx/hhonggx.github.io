Jan 14, 2017 | ubuntu | Hits

1、安装resilio sync
参照下面链接的文章进行安装：
https://my.oschina.net/lieefu/blog/756614
启动
sudo systemctl start resilio-sync
2、配置
参考问答：
http://askubuntu.com/questions/284683/how-to-run-bittorrent-sync
其实就是在终端启动了rslsyncz之后，在浏览器里输入：
http://127.0.0.1:8888/gui/
就能进入相应的配置页面
按页面的依次设置用户名和密码，然后进行登录，再输入邮箱就完成了。

1、switch语句作用的数值类型

在 switch(expr1)中,expr1只能是一个整数表达式或者枚举常量,整数表达式可以是 int 基本类型或 Integer 包装类型,由于,byte,short,char 都可以隐含转换为 int,所以,这些类型以及这些类型的包装类型也是可以的。显然,long 和 String 类型都不符合switch 的语法规定,并且不能被隐式转换成 int 类型,所以,它们不能作用于 swtich 语句中。

3、equals() 和 == 的区别

= =：
（1）==操作符专门用来比较两个变量的值是否相等,也就是用于比较变量所对应的内存中所存储的数值是否相同；
（2）如果一个变量指向的数据是对象类型的,那么,这时候涉及了两块内存,对象本身占用一块内存(堆内存),变量也占用一块内存,例如 Objet obj = new Object();变量 obj 是一个内存,new Object()是另一个内存,此时,变量 obj 所对应的内存中存储的数值就是对象占用的那块内存的首地址。对于指向对象类型的变量,如果要比较两个变量是否指向同一个对象,即要看这两个变量所对应的内存中的数值是否相等。

4、实例变量和静态变量的区别

  • 实例变量：当对象被创建时，才会被分配空间，并且每创建一个对象，就有相对应的实例变量，每个实例变量的空间不一样；

  • 静态变量：当类被加载时就被分配空间，不需要创建对象就可以使用，而不管创建多少个对象，都是用的同一个静态变量。

    5、static方法内部是否可以调用非static方法

    不能，因为static方法的调用不需要创建任何对象，可以直接调用，但是非static方法的调用是依赖对象才能调用的。如果static方法内部调用非static方法，在无对象创建的情况下，是行不通的，所以不能。

    6、Integer 和 int 的区别

  • int 是java提供的8种原始数据类型之一，默认值是0；

    • Integer是java为int提供的封装类，默认值是null;提供了多个与整数相关的操作方法，例如将字符串转换为整数的方法。

      7、Math.ceil(), Math.floor(),Math.round()的区别

  • Math.ceil(): 向下取整数；
  • Math.floor(): 向上取整数；
  • Math.round(): 加0.5后向下取整数；
